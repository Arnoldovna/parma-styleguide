Buttons

Es gibt zwei Button-Varianten. Die Höhe des Buttons orientiert sich an der gewählten Base Fontsize:

Kleine Variante bei Schriftgröße 16px
Große Variante bei Schriftgröße 19px

Der Border-Radius ist 4px. Die Farbigkeit bleibt in beiden Varianten gleich.

Primärbutton

Der Primäre Button ist das Leitelement für den HUK24-Nutzer. Jeder Verlinkung auf eine neue Seite, wird mit dem Primary Button angekündigt und dient dem Nutzer so als verständlichen Guide durch die Webseite.

Große Variante

Kleine Variante

{% include ../_assets/img/btn--primary--small.svg %}

Sekundärbutton

Der Sekundär Button ist das aktivierende Element für den HUK24-Nutzer. Jeder Klick des Nutzers führt eine Aktion direkt auf der aktuellen Seite aus. Während der Primäre Button eine weiterleitende Funktion hat, wird der Sekundär-Button für direkte Aktionen im sichtbaren Bereich des Nutzers verwendet.
